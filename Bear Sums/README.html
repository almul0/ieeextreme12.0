Bear Sums

Time limit: 10880 ms
Memory limit: 264 MB

Mitsos the bear is challenging his brother Vangelis with a mathematical task. Mitsos provides a list of integers LLL and another integer value SSS, then he asks Vangelis to check if there are any two items in the list LLL whose sum is equal to the integer SSS.

Since Vangelis is a confident engineer, he decided to write a program that would do all the computations for him and skip the trouble of thinking. Your task is to help Vangelis write a program that reads the input list LLL and the integer SSS and produces either the solution to the problem or provides an error message when there is no solution available.

Standard input

On the first line there will be an integer number TTT (representing the number of test cases) followed by 222 input lines for each test case:

  • On the first line of each test case, there will be 222 integers SSS and EEE, where SSS is the expected sum and EEE is the number of elements in the list.
  • On the second line, there will be EEE integers separated by a space. Each integer represents an element of the list LLL. The elements are not sorted in any way and some could have the same value. In cases where the number EEE is 000, the second line will be empty.

All values for the elements of list LLL will be in the same range as the value SSS.

Standard output

For each test case you will have to write one line that contains:

  • If there is an unique solution: Write two elements, xxx and yyy of the list LLL, separated by a single space, such that x+y=Sx + y = Sx+y=S and x≤yx \le yx≤y.
  • If there are multiple solutions: Pick the first complete pair that appears on the list and provides the correct sum. Print the two list elements forming this pair in increasing order, as above.
  • If there is no solution: Print the error message !OK.
